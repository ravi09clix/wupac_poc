# Wupas POC

## Client - React App

React with react-chart-js-2 for rendering chart

### Installation

Use the npm to install dependencies.

```bash
cd client
npm i
```

### Usage

```
npm start
```

## Backend - React App

Nodejs with Express

### Installation

Use the npm to install dependencies.

```bash
cd backend
npm i
```

### Usage

```
npm start
```

## Test

Visit to `http://localhost:8080/` to access sites
